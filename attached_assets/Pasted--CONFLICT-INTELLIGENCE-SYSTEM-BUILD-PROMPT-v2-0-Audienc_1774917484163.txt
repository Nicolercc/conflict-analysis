## DESIGN SYSTEM

### Aesthetic Direction

**Editorial Intelligence Platform.** The visual language references serious long-form journalism: Foreign Affairs, The Guardian's investigations desk, academic research portals. Warm off-white backgrounds. Deep navy text. Generous whitespace. Serif display type that signals authority and careful thought. This is where a journalist comes to understand something complex — not to be alarmed, but to be informed.

Every element should feel like it belongs in a well-designed research report, not a live broadcast.

### Animation Principles

Animations should feel like a thoughtful document loading — not a live broadcast. Use:
- **Fade + subtle upward drift** for card entrances (opacity 0→1, translateY 10px→0, 400ms ease)
- **Smooth counter interpolation** for scores (no bouncing, linear ease)
- **Gentle bar fills** for risk meter (1.2s ease-out)
- **Slow typewriter** at 18ms/char — reading pace, not urgency
- NO radar sweeps, NO pulse rings, NO glowing scanlines
- Map markers: simple fade-in with a single subtle ripple — not aggressive pulse animation

### Spatial Composition

- Max content width: 1100px centered with auto margins
- Generous section spacing: 56px between major sections
- Cards: 20px padding, 1px border, 6px border-radius — restrained rounding
- Section labels: small caps, letter-spaced, IBM Plex Mono, above each section
- Horizontal rules (`<hr>`) as section dividers — 1px, var(--border-light)
- Left-aligned text throughout — never centered body copy

## HOOKS

### hooks/useTypewriter.js
```js
import { useState, useEffect } from 'react';

export function useTypewriter(text, speed = 18) {
  const [out, setOut] = useState('');
  const [done, setDone] = useState(false);

  useEffect(() => {
    if (!text) return;
    setOut('');
    setDone(false);
    let i = 0;
    const interval = setInterval(() => {
      i++;
      setOut(text.slice(0, i));
      if (i >= text.length) { clearInterval(interval); setDone(true); }
    }, speed);
    return () => clearInterval(interval);
  }, [text, speed]);

  return { out, done };
}
```

### hooks/useAnimCounter.js
```js
import { useState, useEffect } from 'react';

export function useAnimCounter(to, dur = 1200) {
  const [val, setVal] = useState(0);
  useEffect(() => {
    if (!to) return;
    let start = null;
    const step = (ts) => {
      if (!start) start = ts;
      const p = Math.min((ts - start) / dur, 1);
      setVal(Math.round(p * to));
      if (p < 1) requestAnimationFrame(step);
    };
    requestAnimationFrame(step);
  }, [to, dur]);
  return val;
}
```

## SERVER — index.js

Build a Node/Express server on port 3001.

### POST /api/analyze

Accepts `{ url: string }` OR `{ article: string }`.

1. If `url` provided → call `scraper.js` to extract article text.
2. Run two API calls in parallel with `Promise.all`:
   - `callClaude(articleText)` → structured intelligence JSON
   - `callPerplexity(headline, location)` → verification sources + citations
3. Merge both into a single response and return it.

### callClaude(articleText)

POST to `https://api.anthropic.com/v1/messages`

Headers: `x-api-key`, `anthropic-version: 2023-06-01`, `content-type: application/json`

Body:
```json
{
  "model": "claude-sonnet-4-20250514",
  "max_tokens": 2000,
  "system": "<SYSTEM_PROMPT — see below>",
  "messages": [{ "role": "user", "content": "Analyze this article:\n\n{articleText}" }]
}
```

Parse `data.content[0].text`, strip markdown fences, JSON.parse.

### callPerplexity(headline, location)

POST to `https://api.perplexity.ai/chat/completions`

Headers: `Authorization: Bearer {PERPLEXITY_API_KEY}`, `content-type: application/json`

Body:
```json
{
  "model": "sonar",
  "messages": [
    {
      "role": "system",
      "content": "You are a conflict verification researcher. Search for coverage of this event from NON-WESTERN sources including Al Jazeera, Middle East Eye, teleSUR, The Hindu, Africa Report, CGTN, Xinhua, ANI, Dawn, and regional outlets. Return ONLY valid JSON. No markdown. No preamble. Schema: { \"sources\": [ { \"title\": string, \"url\": string, \"outlet\": string, \"region\": \"Western\" | \"Middle East\" | \"Asia\" | \"Africa\" | \"Latin America\" | \"State Media\", \"summary\": string } ], \"consensus\": string, \"divergence\": string }"
    },
    {
      "role": "user",
      "content": "Find international coverage of: {headline} in {location}. Prioritize non-Western and regional sources."
    }
  ]
}
```

Parse `data.choices[0].message.content`, strip fences, JSON.parse.

## SERVER — scraper.js

```js
import fetch from 'node-fetch';
import * as cheerio from 'cheerio';

export async function scrapeArticle(url) {
  const res = await fetch(url, {
    headers: { 'User-Agent': 'Mozilla/5.0 (compatible; ConflictIntelBot/1.0)', 'Accept': 'text/html' },
    timeout: 10000,
  });
  const html = await res.text();
  const $ = cheerio.load(html);
  $('script, style, nav, header, footer, aside, .ad, .advertisement').remove();
  const text = $('article').first().text() || $('main').text() || $('body').text();
  return text.replace(/\s+/g, ' ').trim().slice(0, 8000);
}
```

If scrape fails, throw: "Could not extract article from this URL. Try pasting the text directly."

### handleAnalyze function

```js
async function handleAnalyze() {
  setLoading(true);
  setError(null);
  setResult(null);
  setActive(false);
  try {
    const body = inputMode === 'url' ? { url } : { article };
    const res = await fetch('/api/analyze', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(body),
    });
    if (!res.ok) throw new Error(await res.text());
    const data = await res.json();
    setResult(data);
  } catch (e) {
    setError(e.message || 'Analysis failed. Please try again.');
  } finally {
    setLoading(false);
  }
}
```

## VISUAL ATMOSPHERE — EDITORIAL, NOT MILITARY

The following design choices are MANDATORY. They replace all dark/military styling from any previous spec.

1. **Background is warm cream** `#F7F4EF` — never black or near-black
2. **All cards are white** with light border — never dark panels
3. **Text is dark on light** — never light on dark
4. **No glow effects, no neon, no scanlines** — these cues belong to broadcast, not research
5. **No blinking LIVE badges** — this is an analytical tool, not a newsroom ticker
6. **Map is parchment-toned** — cartographic reference feel, not a targeting system
7. **Markers settle after animation** — ripple twice then become static. Nothing loops forever.
8. **Risk is communicated through color + clear labeling**, not visual alarm (no red flashing, no sirens, no "CRITICAL" in all-caps with glow)
9. **Section spacing is generous** — the reader should feel they have room to think
10. **Typography does the heavy lifting** — Playfair Display commands authority without theatrical effects

## ERROR HANDLING

- Claude returns invalid JSON → show error card, retry once automatically
- Perplexity fails → still render full Claude results, VerificationPanel shows "Verification temporarily unavailable" in text-muted — never block the full analysis
- URL scraper fails → show friendly error card with suggestion to paste text
- Lat/lng out of bounds → clamp with geo.js, log warning
